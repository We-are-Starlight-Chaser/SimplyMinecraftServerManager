# SimplyMinecraftServerManager(SMSM)

简单、易用的 Minecraft 服务端管理工具。

## 项目简介

SimplyMinecraftServerManager 是一个用于简化和自动化 Minecraft 服务器部署、管理及维护的工具，支持多种服务端类型和常见操作。适合个人玩家、小型团队和社区使用。

## 主要特性

- 一键搭建 Minecraft 服务器
- 支持多种服务端类型（如 Paper、Leaves 等）
- 启动、停止、重启服务器的简单命令
- 自动备份
- 用户及权限管理
- 插件管理
- 简洁的GUI
- 多语言支持(开发中)

## 快速开始

1. 克隆本仓库
   ```shell
   git clone https://github.com/We-are-Starlight-Chaser/SimplyMinecraftServerManager.git
   ```

dotnet 10.0.1 & Visual Studio 2022+
